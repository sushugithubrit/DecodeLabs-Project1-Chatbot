# Rule-Based AI Chatbot

## Overview

This project is a simple AI chatbot developed using Python and rule-based logic.

The chatbot interacts with users by responding to predefined keywords and commands.

## Features

* User input handling
* Rule-based responses
* Interactive conversation flow
* Exit command support

## Technologies Used

* Python

## Project Objective

To understand the fundamentals of chatbot development, decision-making logic, and user interaction using Python.
